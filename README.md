# F1 fantasy results calculator
This is a simple results caculator for F1 Fantasy leagues.
It contains accurate data pulled from OpenF1 

## Notes
This ***WILL NOT*** work during a live session. This includes 30 minutes 
before and after every session. For that you must pay for live access 
from OpenF1 and update the API link with your access code.

## How to use
Just fill in your teams within the Teams.csv file and run the program. 
The results will be output in the terminal in position order.
